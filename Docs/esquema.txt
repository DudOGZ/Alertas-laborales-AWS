Sistema de alertas laborales en la nube - Esquema general

1. Entrada:
   - Lista de páginas web a monitorizar
   - Palabras clave definidas por el usuario

2. Proceso:
   - Función Lambda se ejecuta periódicamente (CloudWatch)
   - Analiza el contenido de las páginas
   - Filtra por palabras clave relevantes

3. Salida:
   - Si hay coincidencias, se envía una notificación por correo (SES)

4. Seguridad:
   - IAM controla los permisos de ejecución y envío
   - Se limita el acceso a los recursos necesarios

Notas:
- El sistema está en fase de diseño
- Se prioriza la discreción técnica en la documentación pública



### ENGLISH ###

# Cloud-Based Job Alert System – General Overview

## 1. Input
- List of websites to monitor  
- Keywords defined by the user

## 2. Process
- Lambda function runs periodically (CloudWatch)  
- Analyzes the content of the websites  
- Filters based on relevant keywords

## 3. Output
- If matches are found, a notification is sent via email (SES)

## 4. Security
- IAM manages execution and email-sending permissions  
- Access to resources is strictly limited

## Notes
- The system is currently in the design phase  
- Technical discretion is prioritized in public documentation
